## 二叉搜索树 - BST
如何按照升序打印二叉搜索树的节点.
```js
var eachBST = function (root) {
    if (!root) return;
    eachBST(root.left);
    console.log(root.val);
    eachBST(root.right);
}
```

如何降序打印二叉搜索树的节点.
```js
var eachBST = function (root) {
    if (!root) return;
    eachBST(root.right);
    console.log(root.val);
    eachBST(root.left);
}
```
