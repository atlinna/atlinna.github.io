### 新增的类语法



+ 属性
  + 使用属性列表的方式进行来描述类中的属性
+ 属性的初始化位置
  + 构造函数中
  + 属性默认值
+ 属性可以修饰为可选的
+ 属性可以修饰为只读的
+ 使用访问修饰符
  + 访问修饰符可以控制类中的某个成员的访问权限
  + public：默认的访问修饰符，公开的，所有的代码均可访问
  + private：私有的，只有在类中可以访问。
  + protected：受保护的
+ 属性简写
  + 直接在构造函数中的参数位置，使用描述符定义属性。
  + 必须是参数，且不做任何处理。
+ 访问器
  + 用于控制属性的读取和赋值
  + 使用 get 读取
  + 使用 set 赋值
