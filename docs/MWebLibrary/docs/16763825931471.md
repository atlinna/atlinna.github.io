# 欢迎使用 MWeb

MWeb 是一款功能很全面的 Markdown 编辑，笔记和发布软件。目前已支持 macOS/iPadOS/iOS 三个平台，并针对每个平台做了深度优化，macOS 版的 MWeb Pro 还有静态网站生成功能。所以，MWeb 算是比较复杂的软件了。因此，开始使用 MWeb 前，建议先阅读使用文档。请使用菜单：“帮助 - 使用文档” 来阅读使用文档。
