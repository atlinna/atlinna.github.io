## transition

`transition` 是一个复合属性包含：

+ Transition-property  								需要过度的属性
+ Transtion-duration                                   动画持续的时间
+ Transition-timing-function                      动画过度的函数
+ Transition-delay                                        延时几秒开始执行动画
