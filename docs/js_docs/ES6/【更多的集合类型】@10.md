## 更多的集合类型

### set
`set` 用于存放不重复的数据集合。
创建方式：`var set = new Set()`

如果传入的数组中有重复数据，回默认去重
```
var arr = [1,1,2,3,4,5,5]
var set = new Set(arr);
console.log(set) // [1,2,3,4,5]
```

#### set操作
+ `add(data)` 向`set`集合的末尾添加数据。（如果数据已存在于 set 集合中，则不会再添加一项。）
  - `set` 集合判断数据相同是使用 `Object.is()` （但是除了 +0 与 -0）
