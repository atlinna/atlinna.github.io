    归并排序

    # 归并排序

    所谓归并排序只是用二叉树的后序遍历思想，将我们的数组分割为左、右两部分，然后进行分别排序 比如现在有这样一个数组[5,4,3,2,1],

    首先我们将数组分为两部分也就是 [5,4,3] 和 [2,1] ，然后，再继续分 [5,4] ,[3] |与| [2],[1]。
我们发现，拆分到最后就是两两比较，最后再通过一些操作，合并拆分后排序好的数据。

    var sort = function (nums) {
    var temp = new Array(nums.length);

    var gsort = function (nums, lo, hi) {
        if (lo == hi) return;
        var mid = Math.floor((lo + hi) / 2)
        gsort(nums, lo, mid);
        gsort(nums, mid + 1, hi);
        merge(nums, lo, mid, hi);
    }


    var merge = function (nums, lo, mid, hi) {
        for (let i = lo; i <= hi; i++) {
            temp[i] = nums[i]
        }
        var left = lo, right = mid + 1;
        for (var i = lo; i <= hi; i++) {
            if (temp[left] > temp[right]) {
                nums[i] = temp[right++];
            } else if (temp[left] < temp[right]) {
                nums[i] = temp[left++];
            } else if (left == mid + 1) {
                nums[i] = temp[right++];
            } else if (right == hi + 1) {
                nums[i] = temp[left++];
            }
        }
    }

    gsort(nums, 0, nums.length - 1)
}
