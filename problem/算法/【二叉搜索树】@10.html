    二叉搜索树 - BST

    # 二叉搜索树 - BST

    根据二叉搜索树的特征(根节点左边的值小于根节点的值，根节点右边的值大于根节点及左边的值) 可以用中序遍历来对 BST 进行遍历。 如何按照升序打印二叉搜索树的节点.

    var eachBST = function (root) {
    if (!root) return;
    eachBST(root.left);
    console.log(root.val);
    eachBST(root.right);
}

    如何降序打印二叉搜索树的节点.

    var eachBST = function (root) {
    if (!root) return;
    eachBST(root.right);
    console.log(root.val);
    eachBST(root.left);
}
