    链表的逆转（逆转链表）

    # 链表的逆转（逆转链表）

    // 逆转链表
function linkedReverse(node) {
  if (!node.next.next) {
    node.next.next = node;
    return node.next;
  } else {
    var ret = linkedReverse(node.next);
    node.next.next = node;
    node.next = null;
    return ret;
  }
}

    这个要怎么想呢？

    首先我们要先找到递归的出口，我们首先是不是要找到最后一个节点，然后让最后一个节点的next 指向我们前一个节点。
